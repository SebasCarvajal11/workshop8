Imágenes de producto (origen para el catálogo)

Coloque aquí sus archivos .avif, .png, .webp, .jpg, .jpeg, .gif o .svg.
El nombre del archivo (sin acentos, en minúsculas) debe contener palabras clave que identifiquen el producto:

  • Audífonos / headphones / auricular → SKU audífonos
  • Silla / chair / ergonomic → SKU silla
  • Laptop / portátil / ultrabook → SKU portátil
  • Monitor / pantalla / 4k → SKU monitor

Al iniciar la API (`pricing-api`) las imágenes se copian automáticamente a:
  • assets/products/
  • web/public/products/

También puede ejecutar manualmente: scripts/sync-product-images.ps1 (importa desde esta carpeta y copia a assets y web/public).
